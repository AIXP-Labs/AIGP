# AIAP Protocol Reference Snapshot

Purpose: provide a small local reference basis for the AIAP package generated
by this AINP example.

External origin hint:
`https://github.com/AIXP-Labs/AIAP-Protocol/tree/main/specification`

This snapshot records only the concepts needed by the example:

- an AIAP package has a human-readable package declaration;
- an agent card describes package identity and capabilities;
- an AISOP-style flow can describe a candidate route;
- generated runtime descriptors must not be treated as execution proof unless a
  real runtime executes them and records independent evidence.

Boundary: this file is a local reference snapshot for generation. It is not the
complete AIAP specification, not a latest-version proof, not a trust root and
not evidence that an AIAP runtime executed the generated package.
