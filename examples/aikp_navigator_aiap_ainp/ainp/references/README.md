# AINP Reference Materials

This optional directory contains generation reference materials for the
`aikp_navigator_aiap_ainp` example.

It may contain protocol/spec snapshots, templates, briefs, style guides,
interface notes, policy excerpts, research notes or other files that informed
the generation plan. These files are not generated content; generated content
lives under the package `content_dir`.

`reference_manifest.json` indexes the local files and their integrity hashes.
Those hashes prove recomputable local integrity only. They do not prove the
files are authoritative, current, legally sufficient or externally trusted.
