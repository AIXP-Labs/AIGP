# Specification

The normative AINP specification lives in
`specification/AINP_Protocol.md`.

The Chinese companion lives in
`specification/AINP_Protocol_cn.md`.

The machine-readable schema index is
`specification/schemas.md`. The canonical schema files themselves live in
top-level `schemas/`.

The lightweight standard-library references live in
`specification/standards/` and are indexed in
[`docs/reference/standards.md`](reference/standards.md). They are reference data
for documentation, dashboards and tooling, not schema mirrors or runtime flows.
