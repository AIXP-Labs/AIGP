```bash
python -m pip install "jsonschema>=4" "pytest>=8"
python -B tests/test_ainp.py
python -B -m pytest -q -p no:cacheprovider -p no:asyncio tests/
```

The test suite uses `AINP_TEST_TMPDIR` when set. Otherwise it prefers `C:\tmp`
on Windows or `/tmp` on POSIX before falling back to the process temp
environment and, only if needed, a repo-local `.ainp_test_tmp` directory. This
keeps restricted runners from failing on an unwritable default temp directory.
